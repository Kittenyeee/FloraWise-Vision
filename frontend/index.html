项目信息

课程项目

本项目是计算机视觉技术与应用课程的大作业， 旨在展示深度学习在图像分类中的实际应用。

技术栈

后端: Python + Flask + PyTorch
前端: HTML5 + CSS3 + JavaScript + Bootstrap
模型: MobileNetV2 + 迁移学习

数据集

使用花卉数据集进行训练，包含5个品种， 共5000+张高质量图像。

性能指标

Top-1 准确率: 92%+
推理时间: ~50ms/张
模型大小: 14MB
